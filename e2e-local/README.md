# 🏠 OPC Agent — 纯本地版 (Ollama)

零成本、零网络、完全本地运行的 AI Agent。

## 前置要求
1. Node.js 18+
2. Ollama (https://ollama.ai)
3. 下载模型: `ollama pull qwen2.5:3b`

## 快速开始
```bash
cd e2e-local
npm install
npm start
```

## 功能演示
- ✅ 多轮对话（带记忆）
- ✅ FAQ Skill 自动匹配
- ✅ 内置工具（文件/时间）
- ✅ DeepBrain 知识存储
- ✅ 知识进化 (evolve)
- ✅ 完全离线，无需 API key
